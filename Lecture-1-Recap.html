R can act like a scientific calculator

Core arithmetic ideas

+
−
×
÷
^

R follows the usual order of operations. Parentheses change the order. Functions like sqrt(), log(), and exp() extend simple arithmetic into mathematical computing.

What matters conceptually

  • Numbers in R are values you can compute with immediately.
  • Many mathematical functions are built in.
  • Large and tiny values may appear in scientific notation.

Example idea: 5.1e3 means 5100.

R is designed for numerical thinking — from simple arithmetic to statistical and scientific calculation.

compute

Start with a value or expression.

→
assign

Give it a name with <-.

Why assignment matters

  • You can reuse results later.
  • You can build more complex calculations step by step.
  • Your workspace becomes a collection of named objects.

Useful mental model

Think of R as a lab bench:

  • object = a labeled container
  • name = how you refer to it
  • assignment = placing something in that container

4 // indexing, subsetting, recycling
Posit

Vectors are useful because you can select and transform parts of them

[ ] means “take pieces”

Use positions to extract one element, many elements, or everything except some elements.

Logical filtering

Conditions create TRUE/FALSE flags that let you keep only matching values.

Recycling rule

When lengths do not match, R may repeat shorter values across a longer vector.

Important mindset

You are usually not looping mentally through values one by one. Instead, you tell R: “apply this operation to the entire object” or “keep only the elements that satisfy this condition.”

Watch out: recycling is powerful, but if lengths do not align nicely it can create warnings and mistakes.

Matrices and arrays are vectors with dimensions

Matrix = 2D structure

A matrix organizes values into rows and columns.

You can extract rows, columns, diagonals, or single cells.

Array = 3D or more

  • Same idea as a matrix, but with extra dimensions.
  • Useful for layered data, repeated blocks, or higher-dimensional structures.
  • Subsetting still follows the same indexing logic.

Two kinds of operations

Element-wise operations act on each entry separately. Matrix algebra uses linear algebra rules, like transpose, multiplication, and inverse.

Key mental shift

A matrix is not just “many numbers.” It has shape — and that shape controls what operations are valid.

6 // non-numeric values
Posit

R also works with logical, character, and factor data

Logical

TRUE and FALSE drive conditions, comparisons, and filtering.

Character

Text values are stored as strings. They are not numbers, even if they look numeric.

Factor

Factors represent categories, often with a fixed set of levels and sometimes a natural order.

Why these types matter

  • Data analysis is not only about calculations.
  • You often need to compare, label, group, and classify values.
  • R’s data types help you express the meaning of the data, not just its appearance.
